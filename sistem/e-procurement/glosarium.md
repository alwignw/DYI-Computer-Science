  Berikut adalah **glosarium (kamus istilah)** lengkap yang umum digunakan dalam sistem e-procurement, mencakup istilah teknis, proses, dan dokumen:

---

## A. Istilah Umum E-Procurement

| Istilah | Definisi |
|---------|----------|
| **E-Procurement** | Proses pengadaan barang/jasa yang dilakukan secara elektronik/digital |
| **E-Sourcing** | Proses pencarian, seleksi, dan evaluasi pemasok secara online |
| **E-Tendering** | Proses tender/lelang pengadaan yang dilakukan secara elektronik |
| **E-Auction** | Lelang harga real-time (reverse auction) melalui platform online |
| **E-Purchasing** | Pembelian langsung melalui katalog elektronik (e-catalog) |
| **E-Invoicing** | Sistem faktur dan pembayaran yang terintegrasi secara digital |
| **Supplier Portal** | Portal khusus bagi vendor untuk mengakses tender dan mengelola profil |
| **Buyer Portal** | Dashboard bagi pembeli untuk mengelola seluruh proses pengadaan |

---

## B. Dokumen & Proses Pengadaan

| Istilah | Singkatan | Definisi |
|---------|-----------|----------|
| **Purchase Requisition** | **PR** | Permintaan pembelian dari user/internal ke departemen procurement |
| **Purchase Order** | **PO** | Dokumen pemesanan resmi dari buyer ke supplier |
| **Request for Quotation** | **RFQ** | Permintaan penawaran harga ke beberapa vendor |
| **Request for Proposal** | **RFP** | Permintaan proposal detail (teknis + komersial) |
| **Request for Information** | **RFI** | Permintaan informasi awal untuk mengenal kemampuan vendor |
| **Tender Document** | - | Dokumen lengkap tender (spesifikasi, terms, kriteria evaluasi) |
| **Bid Bond/Jaminan Penawaran** | - | Jaminan uang dari vendor sebagai komitmen mengikuti tender |
| **Performance Bond/Jaminan Pelaksanaan** | - | Jaminan uang untuk memastikan vendor menjalankan kontrak |
| **Terms of Reference** | **TOR** | Dokumen yang menjelaskan ruang lingkup dan spesifikasi pekerjaan |
| **Bill of Quantity** | **BOQ** | Daftar rincian volume dan harga pekerjaan (umum di konstruksi) |
| **Spesifikasi Teknis** | - | Detail teknis spesifik barang/jasa yang dibutuhkan |
| **Berita Acara** | **BA** | Dokumen catatan hasil rapat/evaluasi tender |
| **Surat Penunjukan Penyedia Barang/Jasa** | **SPPBJ** | Surat resmi penunjukan pemenang tender (khusus pemerintah) |

---

## C. Evaluasi & Seleksi

| Istilah | Definisi |
|---------|----------|
| **Evaluasi Administrasi** | Penilaian kelengkapan dokumen persyaratan (legal, perpajakan, dll) |
| **Evaluasi Teknis** | Penilaian kemampuan teknis vendor terhadap spesifikasi yang diminta |
| **Evaluasi Harga** | Penilaian dan perbandingan harga penawaran dari vendor |
| **Evaluasi Kualifikasi** | Penilaian kemampuan umum vendor (pengalaman, keuangan, SDM) |
| **Passing Grade** | Nilai ambang batas minimal untuk lolos evaluasi |
| **Scoring/Weighting** | Pembobotan kriteria evaluasi (misal: teknis 40%, harga 60%) |
| **Clarification** | Proses klarifikasi penawaran vendor kepada panitia |
| **Negotiation/Negosiasi** | Proses tawar-menawar harga atau terms setelah evaluasi |
| **Best Value** | Pemilihan vendor berdasarkan value terbaik (bukan hanya harga termurah) |
| **Lowest Responsive Bidder** | Pemenang dengan harga terendah yang memenuhi syarat |
| **Disqualification** | Pencoretan vendor karena tidak memenuhi syarat |
| **Blacklist/Daftar Hitam** | Daftar vendor yang dilarang mengikuti tender karena pelanggaran |

---

## D. Jenis-Jenis Pengadaan

| Istilah | Definisi |
|---------|----------|
| **Pengadaan Langsung** | Pembelian langsung tanpa proses tender (nilai kecil/urgent) |
| **Penunjukan Langsung** | Penunjukan vendor tanpa tender (alasan khusus: sole agent, dll) |
| **Tender Terbuka** | Tender yang terbuka untuk semua vendor yang memenuhi syarat |
| **Tender Terbatas** | Tender yang hanya diikuti vendor yang diundang (shortlist) |
| **Seleksi** | Pengadaan jasa konsultan dengan evaluasi kualitas utama |
| **Lelang Terbatas** | Lelang dengan jumlah peserta terbatas |
| **Lelang Umum** | Lelang terbuka untuk umum |
| **Pembelian Secara Elektronik** | Pembelian melalui e-catalog tanpa tender |
| **Sayembara/Design Contest** | Kompetisi desain/rancangan dengan hadiah |
| **Pengadaan Rutin/Repeat Order** | Pembelian berulang dari kontrak sebelumnya |
| **Framework Agreement** | Perjanjian kerangka untuk pengadaan berkala dalam periode tertentu |
| **Spot Buying** | Pembelian spot/insidentil tanpa perencanaan panjang |

---

## E. Kontrak & Legal

| Istilah | Definisi |
|---------|----------|
| **Kontrak/Contract** | Perjanjian hukum antara buyer dan supplier |
| **Service Level Agreement** | **SLA** | Kesepakatan tingkat layanan yang harus dipenuhi vendor |
| **Liquidated Damages** | Denda keterlambatan atau ketidaksesuaian pekerjaan |
| **Force Majeure** | Kondisi keadaan kahar yang membebaskan tanggung jawab kontrak |
| **Termination/Pemutusan** | Penghentian kontrak sebelum jangka waktu berakhir |
| **Amendment/Addendum** | Perubahan/perjanjian tambahan atas kontrak utama |
| **Scope of Work** | **SOW** | Lingkup pekerjaan yang harus dilaksanakan vendor |
| **Delivery Order** | **DO** | Perintah pengiriman barang berdasarkan kontrak |
| **Work Order** | Perintah pelaksanaan pekerjaan/jasa |

---

## F. Pembayaran & Keuangan

| Istilah | Singkatan | Definisi |
|---------|-----------|----------|
| **Invoice/Faktur** | - | Tagihan resmi dari supplier ke buyer |
| **Goods Receipt** | **GR** | Tanda terima barang dari warehouse/logistik |
| **Three-Way Matching** | - | Pencocokan PO, GR, dan Invoice sebelum pembayaran |
| **Purchase-to-Pay** | **P2P** | Siklus lengkap dari pembelian hingga pembayaran |
| **Source-to-Pay** | **S2P** | Siklus lengkap dari sourcing hingga pembayaran |
| **Down Payment/Uang Muka** | **DP** | Pembayaran di muka sebelum pekerjaan dimulai |
| **Termin/Progress Payment** | - | Pembayaran bertahap sesuai progress pekerjaan |
| **Retention Money** | - | Uang jaminan pemeliharaan yang ditahan (5-10%) |
| **Escrow Account** | - | Rekening bersama untuk transaksi dengan nilai besar |
| **Early Payment Discount** | - | Diskon untuk pembayaran lebih cepat dari jatuh tempo |

---

## G. Katalog & Master Data

| Istilah | Definisi |
|---------|----------|
| **E-Catalog/Katalog Elektronik** | Katalog digital berisi daftar barang/jasa dengan harga standar |
| **PunchOut Catalog** | Katalog vendor yang terintegrasi langsung ke sistem buyer |
| **Item Master** | Database master seluruh item/barang yang bisa diajukan |
| **SKU (Stock Keeping Unit)** | Kode unik untuk identifikasi barang |
| **UNSPSC** | Kode klasifikasi produk dan jasa internasional |
| **Preferred Vendor** | Vendor pilihan/prioritas untuk kategori tertentu |
| **Approved Vendor List** | **AVL** | Daftar vendor yang telah disetujui/terverifikasi |
| **Vendor Classification** | Klasifikasi vendor (strategis, operasional, tactical, dll) |
| **Spend Analysis** | Analisis pengeluaran pengadaan untuk optimasi |

---

## H. Teknologi & Sistem

| Istilah | Definisi |
|---------|----------|
| **ERP (Enterprise Resource Planning)** | Sistem terintegrasi (SAP, Oracle, dll) yang mencakup modul procurement |
| **API (Application Programming Interface)** | Antarmuka untuk integrasi sistem e-procurement dengan sistem lain |
| **EDI (Electronic Data Interchange)** | Pertukaran dokumen bisnis secara elektronik antar sistem |
| **Digital Signature/Tanda Tangan Digital** | Tanda tangan elektronik yang sah secara hukum |
| **Timestamp** | Stempel waktu digital untuk membuktikan waktu transaksi |
| **Audit Trail** | Jejak audit lengkap dari setiap aktivitas dalam sistem |
| **Role-Based Access Control** | **RBAC** | Pengaturan hak akses berdasarkan role user |
| **Single Sign-On** | **SSO** | Login sekali untuk mengakses multiple sistem |
| **Cloud-Based/SaaS** | Sistem berbasis cloud (Software as a Service) |
| **On-Premise** | Sistem yang di-install di server internal perusahaan |

---

## I. Metrik & KPI (Key Performance Indicators)

| Istilah | Definisi |
|---------|----------|
| **Procurement Cycle Time** | Waktu siklus pengadaan dari PR hingga PO |
| **Cost Savings** | Penghematan biaya dari negosiasi atau efisiensi |
| **Cost Avoidance** | Penghindaran kenaikan biaya di masa depan |
| **Purchase Order Accuracy** | Tingkat akurasi PO (spesifikasi, jumlah, harga) |
| **Supplier Performance Scorecard** | Penilaian performa vendor secara berkala |
| **Contract Compliance** | Tingkat kepatuhan terhadap kontrak |
| **Spend Under Management** | Persentase pengeluaran yang dikelola procurement |
| **Maverick Buying** | Pembelian di luar sistem/prosedur yang ditetapkan |

---

## J. Istilah Khusus Pemerintah Indonesia (LPSE)

| Istilah | Definisi |
|---------|----------|
| **LPSE** | Layanan Pengadaan Secara Elektronik |
| **SPSE** | Sistem Pengadaan Secara Elektronik |
| **RUP** | Rencana Umum Pengadaan (daftar pengadaan tahunan) |
| **Pokja ULP** | Pokja Unit Layanan Pengadaan |
| **PA** | Pejabat Pengadaan |
| **PPK** | Pejabat Pembuat Komitmen |
| **KAK** | Kerangka Acuan Kerja |
| **HPS** | Harga Perkiraan Sendiri |
| **Pemenang** | Penyedia yang ditetapkan sebagai pemenang tender |
| **Sanggah/Protest** | Keberatan atas hasil pengadaan |
| **Sanggah Banding** | Keberatan atas keputusan sanggah |
