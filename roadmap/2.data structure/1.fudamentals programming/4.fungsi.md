---

**Functions** itu adalah **bagian kode yang punya nama dan tugas tertentu.**
Bayangin aja kayak *alat kecil* dalam program kamu — setiap alat punya fungsi khusus.

Kegunaannya biar kamu **gak perlu nulis kode yang sama berulang-ulang**, cukup panggil fungsi itu kapan pun kamu butuh.
Selain itu, fungsi bikin kode kamu **lebih rapi, teratur, dan gampang dirawat**.

---

Biasanya fungsi itu:

1. **Menerima input** (data dari luar),
2. **Memproses sesuatu** di dalamnya,
3. **Mengembalikan output** (hasilnya).

---

Ada **4 jenis utama fungsi** yang disebut di teks:

1. **Built-in functions**
   → Fungsi yang **sudah disediakan oleh bahasa pemrograman**.
   Contoh:

   * `print()` di Python
   * `len()` untuk menghitung panjang data
   * `console.log()` di JavaScript

2. **User-defined functions**
   → Fungsi yang **kamu tulis sendiri** sesuai kebutuhan.
   Contoh:

   ```python
   def tambah(a, b):
       return a + b
   ```

3. **Anonymous functions (lambda functions)**
   → Fungsi **tanpa nama**, biasanya dipakai kalau cuma butuh fungsi cepat.
   Contoh:

   ```python
   kali = lambda x, y: x * y
   ```

   atau di JavaScript:

   ```javascript
   const kali = (x, y) => x * y;
   ```

4. **Higher-order functions**
   → Fungsi yang **menerima fungsi lain sebagai argumen** atau **mengembalikan fungsi lain**.
   Contoh:

   ```python
   def apply(func, value):
       return func(value)
   ```

---

**Singkatnya:**

> Function = potongan kode yang punya tugas tertentu, bisa dipakai berulang, dan bikin program lebih rapi serta efisien.
