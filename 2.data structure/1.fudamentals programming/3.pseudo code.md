**Pseudo code** (atau **pseudocode**) adalah **cara menulis logika program dalam bentuk bahasa manusia**, bukan dalam bahasa pemrograman sungguhan.
Tujuannya biar kita bisa **menjelaskan langkah-langkah algoritma** dengan mudah tanpa pusing mikirin aturan (syntax) dari bahasa tertentu.

Contohnya gini:
Misalnya kamu mau buat program yang nyari angka terbesar dari dua angka.
Kalau pakai *pseudocode*, kamu bisa nulis:

```
Mulai
  Baca angka A dan B
  Jika A lebih besar dari B
    Tampilkan A
  Jika tidak
    Tampilkan B
Selesai
```

Lihat? Itu belum dalam bentuk kode Python, JavaScript, atau lainnya — tapi kamu langsung paham maksudnya.

Pseudo code ini berguna banget buat:

* **Merancang algoritma** sebelum mulai nulis kode beneran
* **Belajar logika pemrograman** tanpa bingung syntax
* **Diskusi dengan orang lain**, bahkan yang gak terlalu jago ngoding

Analogi gampangnya:

> Pseudocode itu kayak *rencana kerja* sebelum kamu benar-benar bikin program.
> Kamu fokus ke “**apa yang dilakukan**”, bukan “**gimana cara nulisnya**” di bahasa tertentu.

Jadi intinya:
**Pseudo code = tulisan langkah-langkah logika program dalam bahasa manusia, bukan kode asli.**
